```

   Base: neutral-100

   Duration: 1.5s, infinite
   MANDATORY on every loading state — never blank screen

    Push: New slides from right (100%→0%), 280ms
    Pop: Current slides to right (0%→100%), 250ms

    Viya orb appears, scales with pull distance
```
